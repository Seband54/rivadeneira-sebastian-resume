**Tecnologías**: Scrum, Kanban, Jira, Git, CI/CD, Infraestructura en la Nube, Arquitectura MVC, Machine Learning

**Descripción General**: Plataforma web diseñada para la detección temprana del riesgo de deserción estudiantil, desarrollada bajo metodologías Design Thinking y Lean Startup dentro de marcos Scrum y Kanban, implementada utilizando prácticas de ciclo de vida DevOps.

**Logros Clave**:
- **Gestión de Proyectos**: Backlog organizado en Jira con épicas e historias de usuario distribuidas en múltiples sprints
- **Desarrollo Colaborativo**: Implementado con responsabilidad rotativa de Scrum Master y estrategia de desarrollo en ramas de características
- **Control de Versiones e Integración**: Uso de ramas Git con integración continua a rama principal, asegurando calidad de código
- **Desarrollo Backend**: Desarrollo colaborativo de lógica de negocio, autenticación de usuarios y conectividad de base de datos
- **Integración IA/ML**: Implementación de motor de evaluación de riesgo con módulo de aprendizaje automático aplicado a datos académicos
- **Diseño UI/UX**: Creación de prototipos de interfaz para arquitectura Modelo-Vista-Controlador
- **Despliegue en la Nube**: Despliegue continuo en infraestructura en la nube asegurando escalabilidad

**Tecnologías Utilizadas**: Framework Scrum, Metodología Kanban, Gestión de proyectos Jira, Control de versiones Git, Pipelines CI/CD, Plataformas en la nube, Backend Python/JavaScript, Diseño de bases de datos

### Simulación de Sistemas - Optimización de un Carwash mediante Simulación de Eventos Discretos

**Logros Clave**:
- **Recolección de Datos**: Recolección de datos operacionales reales de procesos de carwash

- **Desarrollo del Modelo**: Construcción de modelo de simulación de eventos discretos capturando todos los procesos operacionales
- **Análisis de Escenarios**: Comparación de múltiples escenarios y configuraciones operacionales
- **Evaluación de Resultados**: Evaluación de resultados de simulación utilizando métricas estadísticas

**Impacto**: Identificación de cuellos de botella, proposición de ajustes de personal y estrategias de optimización de equipos resultando en mejoras de eficiencia
